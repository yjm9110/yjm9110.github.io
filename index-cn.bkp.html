📝 个人简介

袁家明于2026年3月获得了俄勒冈大学（University of Oregon）计算机科学专业的博士学位。自2021年起，他一直致力于应用密码学的研究，研究方向涵盖可搜索加密（Searchable Encryption）、属性基加密（Attribute-Based Encryption）以及注册加密（Registered Encryption, RE），旨在提升数据安全性和系统效率。目前，他已在网络安全领域发表了13篇学术论文。

🪧 项目经历

  • 安全移动中心 (Secure Mobile Center)： 参与设计并实现安全移动系统，该项目由新加坡国家研究基金会 (NRF) 资助，资助金额为 6,415,200 新币 (项目编号：NRF2014NCR-NCR001-012)。 [2016年11月 – 2020年2月]
  • 可链接秘密握手的新型构造 (A new construction for linkable secret handshake)： 负责基于 Java 的实验实现。论文发表于 Computer Journal, 2020。 [2020年3月 – 2020年6月]
  • 云端极速且隐私保护的外包计算 (Lightning-Fast and Privacy-Preserving Outsourced Computation in the Cloud)： 负责基于 C 语言和 SGX 的实验实现。论文发表于 Cybersecurity, 2020。 [2019年6月 – 2019年9月]
  • 基于 k 次可追责断言的匿名异步支付通道 (Anonymous Asynchronous Payment Channel from k-Time Accountable Assertion)： 负责基于 Java 的实验实现。论文发表于第18届国际密码学与网络安全会议 (CANS 2019)。 [2019年1月 – 2019年5月]
  • 以患者为中心的动态治疗方案隐私保护强化学习设计 (Privacy-Preserving Reinforcement Learning Design for Patient-Centric Dynamic Treatment Regimes)： 负责基于 C 语言的实验实现。论文发表于 IEEE Transactions on Emerging Topics in Computing, 2019。 [2018年9月 – 2019年2月]
  • 面向云辅助物联网的具有策略保护的轻量级属性基关键词检索 (Lightweight Attribute-based Keyword Search with Policy Protection for Cloud-assisted IoT)： 负责基于 Java 的实验实现。论文发表于 IEEE Conference on Dependable and Secure Computing, 2019。 [2018年5月 – 2018年11月]
  • 基于外包 CP-ABE 的安全云共享系统 (A secure cloud sharing system based on outsourced CP-ABE)： 在 Ubuntu/Windows/Android 平台上使用 C 和 Python 设计并实现了一个基于外包 CP-ABE 的安全云文件共享系统。 [2016年11月 – 2018年8月]
  • 基于外包 CP-ABE 的安全即时通讯系统 (A secure instant messaging system based on outsourced CP-ABE)： 在 Ubuntu/Windows/Android 平台上使用 C 和 Java 设计并实现了一个基于外包 CP-ABE 的安全即时通讯系统。 [2016年11月 – 2018年8月]
  • SecBook，在线安全文档实时协作应用 (SecBook, a secure document real-time collaboration application online)： 使用 Java, JavaScript, HTML 和 CSS 设计并实现了一个安全文档实时协作 Web 应用，并部署于 AWS EC2。 [2016年3月 – 2016年10月]
  • 实创时新（北京）科技有限公司的官方网站： 使用 HTML, JavaScript 和 CSS 为 RTI 设计并实现了中英文双语官方网站，通过 AWS S3 部署。 [2016年3月 – 2016年10月]
